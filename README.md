# 📚 StudyOrganizer

Aplicação web para organização de estudos, permitindo gerenciar tarefas, disciplinas e progresso de aprendizado.

## 🚀 Tecnologias utilizadas

* C# / .NET (API)
* Angular (Front-end)
* SQL Server (Banco de dados)
* Entity Framework

## 💻 Funcionalidades

* Cadastro de tarefas de estudo
* Organização por disciplinas
* Listagem e controle de atividades
* Interface web com navegação por rotas

### Frontend (Angular)

## 📌 Objetivo

Este projeto foi desenvolvido com o objetivo de praticar desenvolvimento full stack utilizando .NET e Angular, simulando uma aplicação real.
